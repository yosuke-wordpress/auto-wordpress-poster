# Golf Ball Radar MVP

ゴルフボール自体に電子部品や飛行特性へ影響する構造を追加せず、スマートフォンのカメラ・GPS・方位センサーで紛失球を探すためのiOS MVPです。

## 採用方式

1. ティー後方にiPhoneを固定し、120/240fpsで打球を撮影
2. Visionで明るい小円形物体を追跡
3. 最後に観測できた画像座標、方位、推定キャリーから着地点と探索半径を算出
4. 「ドラゴンレーダー」風の探索画面で推定地点への方向・距離・精度を表示
5. 草むら付近ではカメラを向け、ボール表面のユーザー固有マークを認識して本人のボールか確認

## 重要な制約

- スマホ単体では、200m先の白い球を着地まで常時追跡することは困難です。
- 本MVPは正確な座標追跡ではなく、探索範囲を数m〜数十mへ狭める支援システムです。
- 市販球への油性ペン、専用スタンプ、印刷済みData Matrix風マークなど、重量・表面形状を実質変更しない識別方法を前提にします。
- 競技利用時は、使用するボール・マーキング・距離計測機能について大会ローカルルールと用具規則の確認が必要です。

## MVP機能

- カメラプレビュー
- ショット開始・停止
- 手動の落下方向補正
- 推定着地点と探索半径
- レーダー表示
- 自分のボールID登録
- 検出履歴
- シミュレーションモード

## Xcodeへの組み込み

1. Xcodeで iOS App / SwiftUI / Swift を新規作成
2. `GolfBallRadar` 以下のSwiftファイルをプロジェクトへ追加
3. Info.plistに以下を追加
   - `NSCameraUsageDescription`
   - `NSLocationWhenInUseUsageDescription`
4. 実機で動作確認

## 次の開発段階

- AVFoundation 240fpsキャプチャ
- Core MLによるゴルフボール専用検出モデル
- カメラ内部パラメータとコース標高を用いた3D弾道推定
- Apple Watchまたは小型三脚による録画操作
- コース地図、OB・池・林の領域表示
- 複数プレイヤーのマーク識別

## 推奨ハードウェア構成

最初はiPhone単体で検証し、精度不足の場合だけティー後方に置く外付け望遠カメラまたは打球レーダーを追加します。ボール側への電子タグ、磁性体、発光体、RFID、UWBは飛行特性・耐衝撃性・規則適合・通信距離の問題が大きいため、初期案では採用しません。
